## Language Toggle Feature

The interactive website includes a language toggle button in the header:
- **Button Location:** Top-right corner of the navigation bar
- **Functionality:** Click to switch between Hebrew (עברית) and English (EN)
- **Persistence:** Language preference is maintained during your session
- **RTL Support:** Hebrew mode automatically enables right-to-left text direction

### Supported Languages:

## Contributing Translations

If you'd like to contribute translations to additional languages, please:

1. Fork the repository
2. Create a new language context in `client/src/contexts/LanguageContext.tsx`
3. Add translation strings for your language
4. Submit a pull request with your changes
